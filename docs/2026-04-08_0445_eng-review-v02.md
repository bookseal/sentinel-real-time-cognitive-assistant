## Key decisions

| Decision | Choice | Why |
|----------|--------|-----|
| Emotion backend | OpenAI Realtime API | Prior torch failure on ARM k3s, image size, memory |
| State management | SessionState class | Per-session isolation, prevents cross-talk |
| SER execution | Fire-and-forget + last-known | Non-blocking, always responsive |
| Model loading | Background thread | App starts instantly |
| VAD fallback | Bypass buffer, fixed 2s timer | Prevents hung pipeline |
| Tests | Full pytest suite | 0% coverage is unacceptable for ML pipeline |
